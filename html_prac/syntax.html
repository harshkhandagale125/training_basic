Syntax of Java

Variables and Data Types

In Java, variables are used to store data. Here's an example of declaring variables and assigning values to them: 

int age = 25;
double salary = 50000.50;
String name = "John Doe";
boolean isEmployed = true;

In the example above, we declared variables of different data types: int for integers, double for floating-point numbers, String for text, and boolean for true/false values.

Control Flow Statements

Java provides several control flow statements to control the flow of execution in a program. Here are some examples: 

if (condition) {
  // code to execute if the condition is true
} else {
  // code to execute if the condition is false
}

for (int i = 0; i < 5; i++) {
  // code to repeat 5 times
}

while (condition) {
  // code to execute repeatedly while the condition is true
}

switch (variable) {
  case value1:
    // code to execute if the variable matches value1
    break;
  case value2:
    // code to execute if the variable matches value2
    break;
  default:
    // code to execute if the variable doesn't match any case
}

The if statement is used for conditional execution, the for loop is used for iteration, the while loop is used for repeated execution, and the switch statement is used for multi-way branching.

Methods

In Java, methods are used to encapsulate a block of code that performs a specific task. Here's an example of a method declaration: 

public int calculateSum(int a, int b) {
  int sum = a + b;
  return sum;
}

In the example above, we declared a method named calculateSum that takes two integer parameters a and b and returns the sum of the two numbers.
